## üöÄ Getting Started

### Clone the Repository

```bash
git clone https://github.com/SatoruZati/capgemini_training.git
cd capgemini_training
```

### Setting Up in Eclipse

1. Open Eclipse IDE
2. Go to `File` ‚Üí `Import` ‚Üí `Existing Projects into Workspace`
3. Select the root directory of the cloned repository
4. Select the specific day's project you want to explore
5. Click `Finish`

### Setting Up in IntelliJ IDEA

1. Open IntelliJ IDEA
2. Select `File` ‚Üí `Open`
3. Navigate to a specific day's directory (e.g., `capgemini_07012026`)
4. Click `OK` to open the project

## üíª Usage

### Running Individual Programs

Each Java file contains a `main` method that can be executed independently:

1. Navigate to the desired Java file in your IDE
2. Right-click on the file
3. Select `Run As` ‚Üí `Java Application` (Eclipse) or click the Run button (IntelliJ)

### Example: Running an Arithmetic Operations Program

```bash
cd capgemini_training/capgemini_10012026/src
javac Package03/arithmetic_operations.java
java Package03.arithmetic_operations
```
